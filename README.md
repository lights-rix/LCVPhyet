# 项目简介

基于深度学习的YOLOv11设备泄漏检测系统，运用先进的计算机视觉技术，实现对设备运行状态的实时监控。系统主要包含功能模块有：YOLOv11目标检测、自定义YOLO数据集训练、用户界面（UI）及登录注册模块。技术特点方面，系统采用了优化的YOLOv11算法，提升检测速度与准确性，同时，用户友好的UI界面与源码开放，便于二次开发与系统集成。此系统专注于提高工业设备安全管理效率，避免潜在的安全风险。

## 核心功能
- YOLOv11目标检测：实现对设备泄漏的实时检测
- 自定义YOLO数据集训练：支持根据特定场景需求进行模型训练
- 用户界面（UI）：提供直观的操作界面
- 登录注册模块：确保系统访问的安全性

## 适用场景
工业设备安全管理与监控，特别适合需要实时监测设备运行状态、及时发现潜在泄漏风险的场景。

## 技术特点
- 采用优化的YOLOv11算法，提升检测速度与准确性
- 用户友好的UI界面设计，便于操作使用
- 源码开放，支持二次开发与系统集成
